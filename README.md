# Chakra-bot

WhatsApp lead qualification bot for GemPundit built on Flask and OpenAI.

Handles incoming messages via ChakraHQ CRM, qualifies leads based on gemstone interest and budget, detects language (Hindi, English, Hinglish, Arabic, Spanish, Italian), converts ratti to carat, scrapes competitor URLs, and routes qualified leads to the sales team.

**Stack:** Python, Flask, OpenAI, Green API, ChakraHQ
